What it does Runtime

Paste the runtime prompt into any reasoning model, then provide biological images or a short observation description.

The runtime keeps observation anchored to biological trace and helps produce consistent AstroBiota field records.

Basic instructions

1. Copy the runtime prompt Paste it into any reasoning model such as ChatGPT or Claude.
2. Provide images or an observation Add one or more biological images and a short observation description.
3. Generate a field record Produce an AstroBiota field record and export the result as a PDF with referenced images attached.
